#### 拆箱管理 Unboxing

Object 类型转换为基本类型

Object 对象在运行中会发生拆箱操作，主要是`toPrimitive`在起作用。对象的三个方法影响`toPrimitive`的返回值。

```js
 var o = {
   toString(){return '2';} //
   valueOf(){return 1} // 加法优先
   [Symbol.toPrimitive](){return 3} // 1
 }
```
